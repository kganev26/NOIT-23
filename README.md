# NOIT-23

1\.Oтворете терминал/команден ред в папката на проекта и изпълнете:



python -m venv venv



2\. Активиране на виртуалната среда

За Windows (CMD): venv\\Scripts\\activate



За Windows (PowerShell): .\\venv\\Scripts\\Activate.ps1



За macOS/Linux: source venv/bin/activate



3\. Инсталиране на зависимости (библиотеки)

След като средата е активирана, инсталирайте необходимите пакети чрез requirements.txt:



pip install -r requirements.txt



4\. Стартиране на приложението

Изпълнете главния файл:



python app.py



След стартиране, отворете браузъра си на адрес: http://127.0.0.1:5000
